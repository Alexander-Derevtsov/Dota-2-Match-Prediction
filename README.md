# Dota-2-Match-Prediction

## Описание проекта

В рамках проекта разработана модель для прогнозирования исходов матчей Dota 2 на основе игровых характеристик команд и игроков.

Основной акцент сделан на подготовке данных, построении информативных признаков и подборе гиперпараметров модели.

---

## Цель проекта

Построить модель машинного обучения для прогнозирования победителя матча и исследовать влияние различных признаков на качество классификации.

---

## Используемые данные

Используется датасет с информацией о матчах Dota 2, содержащий сведения об игроках, героях, игровых событиях и итоговом результате встречи.

---

## Ход исследования

### 1. Подготовка данных

- объединение нескольких таблиц;
- очистка данных;
- обработка пропусков;
- агрегация информации на уровне матча;
- подготовка признаков для обучения модели.

### 2. Feature Engineering

Сформированы признаки на основе игровых характеристик, включая агрегированные показатели команд и игроков.

### 3. Обучение модели

Для решения задачи бинарной классификации использована логистическая регрессия.

Подбор гиперпараметров выполнен с использованием библиотеки **Optuna**.

### 4. Оценка качества модели

Проведена оценка модели на тестовой выборке и анализ влияния гиперпараметров на качество классификации.

---

## Основные результаты

- реализован полный пайплайн подготовки данных;
- выполнено построение информативных признаков;
- проведен автоматический подбор гиперпараметров;
- получена модель для прогнозирования победителя матча.
